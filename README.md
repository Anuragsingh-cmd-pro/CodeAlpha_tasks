# CodeAlpha_tasks
#calculator program
* **Basic Operations**: Seamlessly processes addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
* **Decimal Support**: Uses double-precision floating-point numbers (`double`) for precise decimal output.
* **Error Handling**: Actively blocks division-by-zero operations to prevent program runtime crashes.
* **Input Validation**: Rejects unsupported characters and alerts the user immediately.
* To compile and run this program, you will need a standard C compiler installed on your system:
* **GCC** (Recommended for Linux/macOS)
* **Clang** (macOS)
* **MinGW** (Windows)
